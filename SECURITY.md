# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability, please **do not** open a public issue.

Report privately via [GitHub Security Advisories](https://github.com/SunrisesIllNeverSee/sigrank-vscode/security/advisories/new).

## Scope

- The VS Code extension
- The MCP server bridge
- Local session log reading
- Privacy guarantees (token-only, no message content)

## Out of scope

- The signalaf.com web app (report at [sigrank-app](https://github.com/SunrisesIllNeverSee/sigrank-app/security))
- The sigrank CLI/MCP server (report at [sigrank-mcp](https://github.com/SunrisesIllNeverSee/sigrank-mcp/security))

## Response timeline

- Acknowledgement within 48 hours
- Assessment within 7 days
